Interested in Collaborating with Us?

We would like this project to evolve towards a repository of methods for handling challenging (3D) computer vision problems using Quantum computers. Therefore, we look for contributors and collaborators with great coding and mathematical skills as well as good knowledge in 3D vision, machine learning and quantum computation. If you are interested please send an e-mail to workshop@jackml.com (we can change this for something shared or I can forward this to everyone).
